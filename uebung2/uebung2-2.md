# 2.c.

Downlink-Kanäle = 256 - 1 - 5 - 2 - 32 = 216

Bandbreite / (Übertragungsrate / Bit) = Anzahl Harmonische  
4312,5 Hz / (b / 16) = 32  
4312,5 Hz / 32 = b / 16  
4312,5 Hz / 32 * 16 = b = 2156,25 Bit/s  

Maximaler Downlink = 216 * 2156,25 Bit/s = 465750 Bit/s
